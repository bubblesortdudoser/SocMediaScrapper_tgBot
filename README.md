### Install 

#### 1. Install requirements
```.sh
cd/app
pip install -r requirements.txt
```

#### 2. Rebuild database:db.sqlite
```.sh
./dbmanager rebuild
```

#### 3. Start the app 
```.sh
python bot.py
# or (infinity restart app if it crash)
./restarter.sh
```
